# Adrián & Alexandra — Galería de recuerdos

Galería web mobile-first para que los invitados puedan compartir fotos y vídeos de la boda de Adrián & Alexandra de forma rápida, sencilla y sin iniciar sesión.

## Características

- Subida de fotos y vídeos desde el móvil.
- Galería compartida para invitados.
- Visualización fullscreen.
- Diseño responsive.
- Experiencia pensada para QR.

## Desarrollo local

Abrir `index.html` en el navegador o servir el proyecto con un servidor estático local.

## Nota

Proyecto creado para un evento privado.
